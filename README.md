# 🎬 Movie Recommendation System

A Movie Recommendation System built using User-Based Collaborative Filtering with Cosine Similarity on the MovieLens dataset.

## 📌 Project Overview
This project analyzes user-movie rating data to generate personalized movie recommendations. It uses similarity between users to suggest movies that similar users have highly rated.

## 🚀 Features
- Exploratory Data Analysis (EDA)
- User-Item Matrix Creation
- Cosine Similarity Calculation
- Personalized Movie Recommendations
- Streamlit Web Application Interface

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
